## Iniciando o Projeto

### Pré-requisitos
* React Native

### Instalação

1. Clone o projeto
```sh
git clone https://github.com/felipealvescosta/simple-flatlist.git
```

2. Entre na pasta
```sh
cd flatlist
```

3. Instale as dependências
```sh
yarn install
```
4. Rode o projeto
```JS
yarn ios
```
